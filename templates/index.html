What is this?

Shown below is an explanation for how the project works and what is displayed on the page

This entire project is ran from a python script which is activated by going to a hidden URL path.

This is the equivalent of a demo account, but it is created by Samuel Beaudoin and is instead managed by a trading algorithm (Portfolio Rebalancing).

The account starts with $50,000.

Every 5 days, the portfolio which is displayed in "Current portfolio" is rebalanced (it replaces the 3 worst performing stocks with the best performing stocks of the last 5 days). This means that every 5 days, the data on this page is automatically updated according to what is stored in the database.
